# Mental-Health-Pattern-Monitoring-System
This project presents a Mental Health Pattern Monitoring System, designed to support early  awareness by organizing and analyzing emotional data. The system enables users to log their  daily moods, identify recurring stress triggers, and observe trends in their emotional state.
